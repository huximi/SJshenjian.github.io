20. 有效的括号

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。

有效字符串需满足：
左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

输入: “()[]{}”
输出: true
输入: “(]”
输出: false
输入: “([)]”
输出: false
输入: “{[]}”
输出: true

题解： 利用栈的先入后出特性,遍历至左括号则入栈，右括号则取栈顶元素(此时栈为空则直接false)看是否为对应左括号,
若为是，则继续遍历，若为否，则false
