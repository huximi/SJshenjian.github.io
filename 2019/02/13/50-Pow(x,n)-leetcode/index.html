50_Pow(x,n)_leetcode

2019.02.13 Jian Shen 算法与数据结构  热度 ℃

实现 pow(x, n) ，即计算 x 的 n 次幂函数。

输入: 2.00000, 10
输出: 1024.00000
输入: 2.10000, 3
输出: 9.26100
输入: 2.00000, -2
输出: 0.25000
解释: 2-2 = 1/22 = 1/4 = 0.25
说明:-100.0 < x < 100.0
n 是 32 位有符号整数，其数值范围是 [−231, 231 − 1] 。

题解： 快速幂算法（循环）
x^n次方，n在二进制中可表示为10010形式，即2^4+2^1,那么我们可利用位运算求解
则x^n=x^(2^4) + x^(2^1)
